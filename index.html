About Projects

Lootchadors:

Lootchadors for example is a local multiplayer I made for a client brief as part of a team. Within the team i was team lead, programmer and lead designer.

First Person Shooter Level Generator:

FPS Level Generator was a tool I created for my dissertation for university. The goal of the project was to create a tool to aid in the level design process. The tool gives the user a bunch of options for them to customize to get their desired level layout. The tool randomly generates levels based on these options. After some testing in the field it was concluded that my tool does speed up the process of creating levels and reduces resources needed.
